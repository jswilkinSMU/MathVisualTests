# MathVisualTests
A program to visualize different game programming math techniques including tests for nearest point/point inside, raycasting, 3D shapes, splines and curves, and a pachinko machine.
![Runner Banner](https://github.com/jswilkinSMU/MathVisualTests/blob/main/MVTHeroImg.png)

### How to Use:
    General Controls: 
	    - T to enter slowmo 
	    - Hit the ESC key to quit the game.
	    - F7 goes to next GameMode.
	    - F6 goes to previous GameMode.

    GameNearestPoint:
    	Keyboard Controls: 
    		- ESDF and Arrow keys control the player point.
    		- F8 randomizes shapes
    	Known Issues:
    		- Scaling of infinite line.
    
    GameRaycastVsDiscs:
    	Keyboard Controls:
    		- F8 randomizes discs.
    		- LMB/RMB set ray start/end.
    		- ESDF moves ray start.
    		- IJKL moves ray end.
    
    GameRaycastVsLinesegments:
    	Keyboard Controls:
    		- F8 randomizes lines.
    		- LMB/RMB set ray start/end.
    		- ESDF moves ray start.
    		- IJKL moves ray end.
    
    GameRaycastVsAABB2D:
    	Keyboard Controls:
    		- F8 randomizes boxes.
    		- LMB/RMB set ray start/end.
    		- ESDF moves ray start.
    		- IJKL moves ray end.
    		- Press V to snap ray vertically.
    		- Press H to snap ray horizontally.

    Game3DTestShapes
    	Keyboard Controls:
    		- F8 randomizes shapes.
    		- R toggles between solid and wireframe rasterizer modes.
    		- WASD moves camera horizontally.
    		- QE moves camera vertically.
    		- Space locks raycast and reference position.
    		- LMB grabs and sets down objects.

### Build and Use:
	1. Download and Extract the zip folder.
	2. Open the Run folder.
	3. Double-click Doomenstein_Release_x64.exe to start the program.
